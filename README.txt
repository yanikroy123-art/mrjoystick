JoystickPro - Web App (Demo)
----------------------------------
Arquivos incluídos:
- index.html        (front-end)
- styles.css        (estilos)
- script.js         (front-end JS)
- contact.php       (processa formulário de contacto)
- repair_request.php(processa pedido de reparação)
- messages.log      (criado automaticamente quando mensagens são enviadas)
- repairs.log       (criado automaticamente quando pedidos são enviados)

Como testar:
1. Copie os arquivos para um servidor com suporte PHP (ex: Apache + PHP).
2. Aceda a index.html via o servidor (ex: http://localhost/joystickpro/index.html).
3. Os formulários POST enviarão para contact.php e repair_request.php que gravam em logs.
4. Substitua a lógica de gravação por envio de e-mail ou integração com base de dados em produção.

Cores: Roxo (#7b3bd9) e preto/escuro, layout responsivo.
